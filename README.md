## 🪄 주요 기능과 로🪄 주요

### :one: 사용자

- 회원가입: 이름, 이메일, 비밀번호, 나이, 직업 데이터를 받아 사용자를 등록한다.
- 로그인: 이메일과 비밀번호를 입력해 사용자 정보를 불러온다.

### :two: 은행 상품

- 전체 은행 상품 페이지
- 맞춤형 상품 페이지
- 찜한 상품 페이지
- 장바구니
